# 💳 Credit Card Fraud Detection Dashboard

## 📊 Project Overview
This project focuses on analyzing credit card transaction data to identify fraudulent transactions using data analysis and visualization techniques. The goal is to understand fraud patterns and provide meaningful insights.

---

## 🛠 Tools & Technologies
- Python (Pandas, NumPy)
- Power BI
- Data Visualization

---

## 📌 Key Insights
- Fraud transactions are very rare (~0.17% of total data)
- The dataset is highly imbalanced
- Fraud transactions show different behavior compared to normal transactions

---

## 📈 Dashboard Features
- Total Transactions
- Fraud Transactions
- Fraud Percentage
- Fraud vs Normal Transactions (Count Comparison)
- Average Transaction Amount (Fraud vs Normal)

---


---

## 🚀 Conclusion
This project helps in identifying fraud patterns and supports data-driven decision-making using visual analytics.
